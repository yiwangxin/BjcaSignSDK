# 数字医信SDK
医网信通过⾝份认证和电⼦签名保证医疗体系的信息安全。在⽤户的使⽤过程中，需要经历⾝份审核、证书下载流程后，才能够使⽤签名业务。  
可以参考医网信官方集成文档集成[官方集成文档集成](https://doc.51trust.com/#/%E4%BA%91%E7%AD%BE%E5%90%8D/mobile/ios)
